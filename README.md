# 📩 Multilingual Spam Detector

## 📌 Project Overview

The Multilingual Spam Detector is a Machine Learning-based web application that detects whether a message is **Spam** or **Ham (Not Spam)**.

It supports multiple languages:
- 🇬🇧 English
- 🇮🇳 Hindi
- 🇮🇳 Marathi

The application is built using Python, Scikit-learn, NLTK, and Streamlit.

---

## 🚀 Features

- Spam Detection using Machine Learning
- English, Hindi and Marathi Support
- Interactive Streamlit Web Application
- User-friendly Interface
- Fast Predictions

---

## 🛠️ Tech Stack

## 🤖 Machine Learning Models Used

- Multinomial Naive Bayes
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest

---

## 📊 Model Performance

| Model | Accuracy |
|--------|----------|
| Multinomial Naive Bayes | 95.11% |
| Logistic Regression | 95.02% |
| Support Vector Machine (SVM) | **96.44%** |
| Random Forest | 96.13% |

Best Model: **Support Vector Machine (SVM)**
